# Diagnosis Skill

## Goal

Validate diagnosis data by checking ICD mapping correctness and identifying mutually exclusive diseases.

## Skills
- `check_icd`
- `check_exclusive_disease`

## Output
The agent must output a valid JSON object containing the compiled results of the combined diagnosis checks.

```json
{
  "status_code": 0,  // 0: Success, 1: Warning, 2: Error
  "status_message": "success", 
  "data": {
    "summary": {
      "total_warnings": 0,
      "total_success": 0
    },
    "warnings": [
      {
        "type": "icd_missing | icd_mismatch | excluded_diagnosis",
        "diagnosis_name": "string",
        "suggested_icd": "string",
        "message": "string"
      }
    ],
    "success": [
      {
        "type": "icd_valid | coverage_approved",
        "diagnosis_name": "string",
        "icd": "string",
        "message": "string"
      }
    ]
  }
}
```

The output MUST contain the original status flags, summaries, warnings, and success details generated by the underlying skills. Ensure the output is STRICTLY valid JSON with no preamble or commentary, and that the final messages are in VIETNAMESE.
